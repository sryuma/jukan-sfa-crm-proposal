フェーズ別の詳細

Phase 0：営業フェーズ（6/24 〜 7/1 / 1週間）

  • ヒアリング（30〜60分・1回）
  • 剣サイドで要件整理・ツール比較・概算見積
  • 提案書ドラフト → 7/1（火）橋本社長へ提出
  • 7/10（木）Goofy本提案と比較・判断 → 7/11 契約締結
成果物：ヒアリングシート記入版／提案書／概算見積／契約書ドラフト

Phase 1：要件定義・設計（7/11 〜 7/24 / 2週間）

  • ツール最終選定（Salesforce vs HubSpot vs kintone等の比較検証）
  • 業務フロー棚卸し（リード→商談→受注→施工→アフター）
  • OB顧客20万件のクレンジング設計・名寄せルール策定
  • 要件定義書のレビュー・承認
成果物：ツール選定報告書／業務フロー図／要件定義書／データ移行計画書

Phase 2：CRM/SFA基盤構築（7/25 〜 8/24 / 1ヶ月）

  • CRM/SFA環境構築・初期セットアップ
  • OB顧客20万件のデータ移行（クレンジング → インポート → 検証）
  • Andpad / Freee API連携（1回入力で全体に流れる構成）
  • UAT実施・受入テスト
成果物：CRM/SFA本番環境／データ移行済DB／API連携／UAT合格報告

Phase 3：MA・自動化実装（8/25 〜 10/9 / 1.5ヶ月）

  • 掘り起こしシナリオ設計（設置後10年/20年タイミングでの自動アプローチ）
  • MA自動配信構築（メール／SMS／LINE）
  • 営業録音→CRM保存／同意管理ワークフロー
  • 初回キャンペーン実行・効果測定
成果物：MAシナリオ／自動配信／録音・同意管理／初回キャンペーンレポート

Phase 4：AI活用機能実装（10/10 〜 11/9 / 1ヶ月）

  • 営業録音の文字起こし・ベクトル化
  • トップ営業の勝ちパターン抽出AI
  • 提案サジェストAI（誰に・いつ・何を・どう）
  • 全機能本番リリース・社員教育
成果物：AI分析ダッシュボード／提案サジェスト機能／運用マニュアル／社員向け研修資料

Phase 5：運用伴走（11/10 〜 継続）

  • 月次定例ミーティング（KPIレビュー・改善提案）
  • AI精度向上のための継続学習・運用最適化
  • 剣のAI研修と連動した社員教育・ナレッジ蓄積
  • 3年で売上100億を実現するための戦略支援
成果物：月次レポート／改善提案／継続的なAI精度向上

リスクと留意点

R1：OB顧客データのクレンジング工数が想定を超える可能性 → ヒアリング時にサンプルデータを共有いただき、スケール感を事前確認
R2：Salesforce継続要否の意思決定遅延 → 7/11契約締結直後の要件定義最初の論点として優先確定
R3：Andpad/Freee APIの仕様制約 → Phase 1の最終週に技術調査を完了させ、Phase 2前にリスク確定
R4：営業現場の運用定着 → 剣のAI研修と連動させ、月次定例で運用改善を継続（伴走の本領）
